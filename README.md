# CV Analyzer 

A Django web application that analyzes resumes using AI and provides professional HR-level feedback.

🌐 **Live Demo:** [cvanalysis.pythonanywhere.com](https://cvanalysis.pythonanywhere.com)

---

## Features

- User authentication (register, login, logout)
- Upload PDF resumes
- AI-powered resume analysis using LLaMA 3 (via Groq API)
- Feedback on strengths, weaknesses, and suitability for software engineering roles
- Stores analysis history per user

---

## Tech Stack

- **Backend:** Django 6
- **AI:** Groq API (LLaMA 3.3 70B)
- **PDF Parsing:** PyPDF2
- **Database:** SQLite
- **Deployment:** PythonAnywhere

---
## Usage

1. Register or log in
2. Upload your resume as a PDF file
3. Get instant AI feedback on your resume
